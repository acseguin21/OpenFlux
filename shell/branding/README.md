# Branding Assets

Assets for **OpenCode by VibeCoders United** (Scarlet & Jade theme).

- **Icons** – App icon for macOS (`.icns`), Windows (`.ico`), Linux (PNG). Use Scarlet `#C41E3A` and Jade `#0D9488` per `docs/BRANDING.md`.
- **product.json** – Snippet or full overrides for `product.name`, `product.winx64AppUserDataFolderName`, etc., so the built app identifies as “OpenCode”.
