# OpenCode Core Dependencies

## Python Backend (core/)
The backend handles indexing, RAG, and agent orchestration.

```bash
# Core AI & LLM Connectivity
openai>=1.0.0
anthropic>=0.20.0
ollama>=0.1.0
google-generativeai>=0.4.0

# Context & RAG
tree-sitter>=0.21.0
tree-sitter-languages>=1.10.0
lancedb>=0.6.0  # Vector DB (Native Rust, high performance)
pyarrow>=15.0.0 # Requirement for LanceDB
tantivy>=0.22.0 # Full-text search engine (Native Rust)

# Utilities
pydantic>=2.0.0
fastapi>=0.110.0
uvicorn>=0.27.0
python-dotenv>=1.0.0
```

## Node.js Extension (extensions/opencode-ai-tools/)

## Shell / Build Tools
Requirements for building the VSCodium shell.

- **Node.js**: v18 or v20
- **Python**: 3.10+
- **Yarn**: 1.22+
- **C++17 Compiler**: (GCC, Clang, or MSVC)
