# OpenFlux Scarlet & Jade

Scarlet and Jade accent colors with neutral tones. Part of **OpenFlux by VibeCoders United**.

- **Scarlet** `#C41E3A` — Actions, warnings, emphasis
- **Jade** `#0D9488` — Success, agent, indexing
- **Neutrals** — Stone palette for background and UI

Select: **File > Preferences > Color Theme > OpenFlux Scarlet & Jade (Dark)** (or Command Palette: *Preferences: Color Theme*).
